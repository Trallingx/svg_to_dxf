# Contributing

Thanks for contributing.

## Local Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements-dev.txt
```

## Run App

```powershell
python app.py
```

## Run Tests

```powershell
python -m pytest -q
```

## Pull Requests

- Keep changes focused and small.
- Add or update tests for behavior changes.
- Update `README.md` when adding features or settings.
